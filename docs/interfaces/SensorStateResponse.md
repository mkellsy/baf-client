# Interface: SensorStateResponse

Stores information about the current state of a sensor device.

## Properties

### humidity

> **humidity**: `number`

The current relative humidity.

***

### id

> **id**: `string`

The id of the sensor device.

***

### temperature

> **temperature**: `number`

The current temperature in celsius.
